wsdl:soapVersion (wsdlTable, includeDot)
Obtém a versão do SOAP suportada pelo Web Service

Parameters

  • wsdlTable: Tabela lua gerada a partir do parse do conteúdo do WSDL
  • includeDot: Se true, inclui o ponto no número da versão. O parâmetro é opcional e o valor padrão é true.

Return value:

Retorna 1.1 para SOAP 1.1 ou 1.2 para SOAP 1.2
