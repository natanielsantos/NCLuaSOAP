main.drawText (text, flush, clear, x, y, color)
Desenha uma mensagem na tela

Parameters

  • text: Texto a ser desenhado
  • flush: Se é pra dar um flush no canvas, e exibir as alterações feitas nele. O valor padrão é true.
  • clear: Indica se a área do canvas deve ser limpa ou não. O valor padrão é true
  • x: Posição horizontal onde o texto deve ser impresso
  • y: Posição vertical onde o texto deve ser impresso
  • color: Cor da fonte a ser utilizada

main.paintCloseBtn (x, y)
Exibe o botão fechar, calculando as coordenadas para que ele fique na parte inferior direita da área do canvas. A função não realiza um canvas:flush, que deve ser executado posteriormente quando desejado.

Parameters

  • x: Posição horizontal do botão. Se omitido, será colocado no canto esquerdo
  • y: Posição vertical do botão. Se omitido, será colocado no canto inferior
